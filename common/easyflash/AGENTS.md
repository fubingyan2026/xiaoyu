# modules/easyflash/

嵌入式系统磨损均衡Flash存储管理。为校准数据和配置参数提供持久化存储。

## 结构

```
easyflash/
├── inc/              # 公共头文件
├── src/              # 核心实现
├── port/             # 平台特定端口
└── plugins/          # 可选插件 (JSON等)
```

## 查找位置

| 任务 | 文件 | 说明 |
|------|------|-------|
| Flash API | `inc/easyflash.h` | 主接口函数 |
| 存储管理 | `src/ef.c` | 核心Flash操作 |
| 磨损均衡 | `src/ef_utils.c` | Flash块管理 |
| 端口配置 | `port/` | STM32特定实现 |
| JSON支持 | `plugins/` | 可选数据序列化 |

## 约定

- **磨损均衡**: 自动Flash块轮换
- **错误处理**: 所有操作返回代码
- **原子操作**: 安全写入/擦除序列
- **掉电保护**: 数据完整性保证

## 反模式 (EASYFLASH)

- **不要** 频繁写入Flash (有限写入周期)
- **绝不** 假设写入成功 - 始终检查返回代码
- **避免** 大量写入 (使用基于块的方法)

## 独特风格

- **环境变量**: 键值存储接口
- **自动备份**: 可靠性冗余存储
- **恢复**: 掉电安全操作

## 注意事项

- 用于电机校准数据持久化
- 支持原始数据和结构化存储
- STM32端口配置为内部Flash
- 有限写入耐久性 - 谨慎使用
