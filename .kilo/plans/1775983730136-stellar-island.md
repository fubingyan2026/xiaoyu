## Additional Optimization: easyflash 日志函数简化
Lines 264-315 存在重复代码：三个日志函数都执行相同的 vsnprintf + DEBUG_LOG* 模式。

### 优化方案
在 `ef_port.c` 文件顶部添加静态内联辅助函数：

```c
static inline void ef_log_helper(debug_level_t level, const char *tag, const char *format, va_list args)
{
    char msg_buf[256];
    vsnprintf(msg_buf, sizeof(msg_buf), format, args);
    debug_log(level, tag, "%s", msg_buf);
}
```

然后简化三个日志函数为一行调用：

```c
void ef_log_debug(const char *file, const long line, const char *format, ...)
{
#if PRINT_DEBUG
    va_list args;
    va_start(args, format);
    ef_log_helper(DEBUG_LEVEL_DEBUG, "easyflash", format, args);
    va_end(args);
#endif
}

void ef_log_info(const char *format, ...)
{
    va_list args;
    va_start(args, format);
    ef_log_helper(DEBUG_LEVEL_INFO, "easyflash", format, args);
    va_end(args);
}

void ef_print(const char *format, ...)
{
    va_list args;
    va_start(args, format);
    ef_log_helper(DEBUG_LEVEL_INFO, "easyflash", format, args);
    va_end(args);
}
```

这样将每个函数的 7 行代码减少到 5 行，消除重复的 buffer 声明和 vsnprintf 调用。
